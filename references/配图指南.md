# 艾利特文章配图指南（智谱 GLM-Image · B2B 科技信息海报风格）

本文档规定艾利特 SEO/GEO 文章的配图规则：每篇文章 3-4 张图（1 封面 + 2-3 正文），统一 **B2B 科技信息海报风格**（3D 写实场景 + 信息卡片文字排版），由智谱 GLM-Image 生成。写作部分仍由 `写作提示词.md` 主导，配图是文章完成后的独立增强环节。

---

## 核心原则：让图片承担信息，而非只是装饰

智谱 GLM-Image 的核心能力是「**读懂指令、写对文字**」，采用「自回归 + 扩散解码器」混合架构，专门克服海报、PPT、科普图等**文字密集**场景的生成难题（词准确率 0.91）。因此艾利特配图必须**带文字**：

- **封面图** = 文章标题的可视化海报（大标题 + 副标题 + 卖点图标）
- **正文配图** = 段落主题的可视化解析（小标题 + 关键标注）

**禁止生成无文字的纯场景插画**——那既浪费 GLM-Image 的文字渲染能力，又缺乏「成品感」，读起来像概念草图而非企业官网配图。

## 一、配图规划

### 1. 数量与角色

| 角色 | 数量 | 尺寸 | 内容构成 |
|---|---|---|---|
| 封面图 | 1 张 | `1536x864`（16:9） | 左侧信息卡片（标题+副标题+卖点图标）+ 右侧 3D 场景 |
| 正文配图 | 2-3 张 | `1024x1024`（正方形） | 3D 场景主体 + 左上角小标题 + 关键部位标注 |

合计 3-4 张。**不要超过 4 张**。

### 2. 配图位置

- **封面图**：SEO 信息块之后、`## 文章内容` 引言段之前。
- **正文配图**：嵌在核心 H2 段落正文之后（产品选型、工艺解析、行业应用、技术原理段落）。
- **最后一张正文图**：总结段之前。

### 3. 什么段落适合配图

✅ 适合：码垛 / 焊接 / 装配 / 检测工序场景、人形机器人双臂操作、力控精密操作、工厂产线 / 无尘车间 / 仓储物流环境。

❌ 不适合（用文字/表格）：纯参数对比、内链清单、FAQ、数据罗列。

## 二、视觉风格规范（B2B 科技信息海报）

### 1. 风格定义

- **画种**：B2B 科技信息海报 = 高精度 3D 写实场景 + 信息卡片文字排版
- **构图**：场景区 + 文字区分区，视觉层次清晰，留白约 30%
- **色调**：科技蓝 `#0078D7` + 工业灰 `#F0F0F0` + 棕色（纸箱/工件）+ 白色，冷调为主
- **质感**：3D 写实渲染、真实光影、金属 / 哑光塑料 / 牛皮纸材质
- **字体**：无衬线（思源黑体风）；大标题粗体深蓝 `#002060`，副标题/标注深灰 `#333333`

### 2. 封面图布局（参考标杆）

- **右 60%**：3D 写实主场景（机器人 + 工件 + 环境 + 光影）
- **左 40%**：科技蓝→白渐变信息卡片
  - 顶部：**大标题**（文章核心标题，8-12 字，粗体深蓝，字号最大）
  - 中部：**副标题**（3-4 关键词用 · 分隔，深灰）
  - 底部：**3 个圆形图标 + 4 字卖点标签**（横排）

### 3. 正文配图布局

- **主体 70-80%**：3D 写实场景
- **左上角/顶部**：半透明科技蓝信息条 + **小标题**（对应 H2，6-10 字，白色粗体）
- **关键部位**：细引导线 + **中文标注**（参数/部件名，深灰细字，最多 3 条）

### 4. 必须做到

- ✅ 图片带**正确中文文字**（标题/副标题/标注）—— GLM-Image 的强项，要充分利用
- ✅ 文字与背景对比 ≥ 4:1（深字浅底，或白字蓝底）
- ✅ 视觉层次：场景 > 大标题 > 副标题/标注
- ✅ 留白充足，不拥挤

### 5. 必须避免

- ❌ **无文字纯场景插画**（浪费模型能力、无成品感）
- ❌ 单张图文字过多（>20 字）导致排版混乱
- ❌ 真实人脸特写、暴力危险姿态、可识别的竞品 logo
- ❌ 把 AI 插画伪装成艾利特型号实拍（来源红线）
- ⚠️ 智谱返回的图右下角会自带「AI生成」水印——属国内 AI 内容合规标注，发布时可保留（合规）或用图片编辑器裁剪，**不要**试图用 prompt 去除（平台强制）

### 6. 尺寸与像素规范

智谱 GLM-Image size 约束：长宽 32 的整数倍、单边 512–2048、总像素 ≤ 2²²。

| 用途 | size | 校验 |
|---|---|---|
| 封面图（16:9 横版） | `1536x864` | 1536÷32=48 ✓，864÷32=27 ✓ |
| 正文配图（1:1） | `1024x1024` | 1024÷32=32 ✓ |
| 竖版封面（可选） | `864x1536` | 同上 ✓ |

## 三、文字内容规则

### 1. 封面图文字（必填）

| 层级 | 内容 | 来源 | 字数 |
|---|---|---|---|
| 大标题 | 文章核心标题 | 写作提示词产出的 SEO Title 精简版 | 8-12 字 |
| 副标题 | 3-4 个文章关键词 | 文章核心维度 | 用 `·` 分隔 |
| 卖点图标 ×3 | 文章核心卖点 | 文章亮点提炼 | 每个 4 字 |

**示例**（码垛机器人厂家主题）：
- 大标题：「码垛机器人厂家有哪些」
- 副标题：「主流品牌 · 选型维度 · 码垛方案」
- 卖点：「多家品牌」「高效稳定」「快速换产」

### 2. 正文配图文字

| 层级 | 内容 | 字数 |
|---|---|---|
| 小标题 | 对应 H2 段落主题 | 6-10 字 |
| 标注（可选） | 场景关键部件/参数 | 每个 4-6 字，最多 3 条 |

## 四、Prompt 构造规则（GLM-Image 最佳实践）

### 1. 语言：中文 prompt

GLM-Image 中文理解强、文字渲染准。**用中文 prompt**（描述场景 + 要写的文字），模型既能理解场景又能正确渲染中文标题。经实测，中文大标题、副标题、图标文字均可精准呈现。

### 2. 文字用中文引号 "" 包裹（关键技巧）

GLM-Image 对引号内文字精准渲染。**所有要出现在图上的文字，prompt 里用中文双引号 `""` 包裹**，如 `"码垛机器人厂家有哪些"`。

### 3. 分区域结构化描述

封面按「右侧场景区 + 左侧文字卡片区」分别描述；正文按「主体场景 + 左上角标题 + 关键标注」描述。

### 4. 明确文字属性

字体（粗体无衬线）、颜色（深蓝 `#002060` / 深灰 `#333333` / 白）、字号对比（大标题最大）、位置（顶部 / 左上 / 底部）。

### 5. 明确配色与质感

科技蓝 + 工业灰 + 棕色 + 白色；3D 写实、真实光影材质、高精度。

### 6. 封面 Prompt 模板

```
一张高质量的B2B科技海报风格配图，16:9横版构图。右侧约60%画面是[3D场景描述：机器人+工件+环境+光影材质]。左侧约40%画面是科技蓝到白色的渐变信息卡片区域：顶部是大标题中文"[文章标题]"，粗体无衬线字体，深蓝色，字号最大；标题正下方是副标题中文"[关键词1 · 关键词2 · 关键词3]"，中等字号，深灰色；卡片底部横向排列三个圆形小图标，每个图标下方分别配中文文字"[卖点1]""[卖点2]""[卖点3]"，小字号深灰色。整体配色以科技蓝、工业灰、棕色、白色为主，专业克制，留白充足，视觉层次清晰，像工业机器人企业官网的封面主视觉。不要水印，不要任何品牌logo。
```

### 7. 正文配图 Prompt 模板


## 六、alt 文案与图注规范（SEO / GEO）

图片**本身已有文字**（标题/标注），alt 与图注是 Markdown 文本层，与图内文字**互补**，为搜索引擎和生成式搜索提供可索引文本（图内文字模型看不见，alt/图注是 SEO 入口）。

### 1. alt 文案
- 包含核心关键词 + 图片内容描述，中文 15-30 字。

✅ `![码垛机器人在物流仓库执行纸箱码垛作业](url)`
❌ `![图片1](url)`

### 2. 图注
- 紧跟图片下一行，斜体 `*图N：说明*`，编号连续，一句话说明。

## 八、配图质量自检

生成并嵌入前自检：

- [ ] 图片是否**带正确中文文字**（封面：标题+副标题+卖点；正文：小标题+标注）？
- [ ] 要出现在图上的文字，是否都用中文引号 `""` 包裹写进 prompt？
- [ ] 是否 **B2B 科技信息海报风格**（3D 场景 + 信息卡片），而非无文字插画？
- [ ] 配色是否科技蓝 + 工业灰 + 棕色 + 白？
- [ ] 文字与背景对比是否 ≥ 4:1？
- [ ] 单张图文字是否 ≤ 20 字（避免排版混乱）？
- [ ] 数量是否 3-4 张（封面 1 + 正文 2-3）？
- [ ] 封面 `1536x864`、正文 `1024x1024`？
- [ ] 配图主体是否跟随写作提示词第四节产品匹配结论？
- [ ] alt 是否含核心关键词？图注是否编号连续？
- [ ] 是否避免了把 AI 插画伪装成艾利特型号实拍（来源红线）？
- [ ] 文末是否附了「配图说明」注释块？
